Question-1: What are the different ways to select an element in the DOM?

There are some way to select an element for example -

  1. getElementById()
  2. getElementsByClassName()
  3. getElementsByTagName()
  4. querySelector()
  5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: It returns all content with HTML.
innerText: It returns only human readable text which are visible in browser, is considering css style.
textContent: It returns every element in the node, does not consider css style.

Question-3: What is event delegation in the DOM?

It's a technic where a event listener is added in parent element, it can handle if a event trigger in child.

Question-4: What is event bubbling in the DOM?

When a event has been triggered in child, parent event has been triggered as well.

Question-5: How do you create, add, and remove elements using JavaScript?

let newDiv = document.createElement("div"); // Create new element
newDiv.innerText = "Hello, I am new!"; // Set innerTest
document.body.appendChild(newDiv); // Add/Append in body element
document.body.removeChild(newDiv); // Remove
